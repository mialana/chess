# Chess in Java

![img](./files/thumbnail.png)

**[Demo Video](https://youtu.be/xzkEUZbMlUU)**

### Overview

A simple chess application with state-saving and move-hints / limitations.

## Prerequisites

Install Maven onto your system.

```bash
# on Mac
brew install maven
```

### Build instructions

In the project directory: 
```bash
make compile
```

```bash
make run
```
